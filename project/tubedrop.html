TubeDrop

Удобный клиент для быстрой загрузки медиаконтента с YouTube

C#
WPF
.NET

> Об утилите

TubeDrop — это Windows-приложение, созданное для того, чтобы скачивать видеоролики с YouTube в пару кликов без необходимости открывать браузер или использовать перегруженные рекламой веб-сервисы.

В будущем планируется реализация выбора качества видео, папки сохранения и скачивания только аудиодорожки.

Для загрузки видео из России требуется обход блокировки YouTube.

> Системные требования

ОС: Windows 10 (1809+)
Библиотека: .NET 10.0+
