UWP-Remo

Специализированная программа для очистки Windows. Удаляет встроенные UWP приложения.

C#
WinForms

> Об утилите

UWP-Remo — это специализированное Windows-приложение, созданное для быстрой и безопасной очистки операционной системы от предустановленного мусора и неиспользуемых встроенных UWP-приложений в один клик.

В будущем планируется интеграция всего функционала UWP-Remo в качестве отдельного модуля в состав комплексного инструмента Tweaker.

Для корректной работы утилиты требуются права администратора.

> Системные требования

ОС: Windows 8.1+
Библиотека: .NET 7.0+
