Tweaker

Инструмент для тонкой настройки, оптимизации и кастомизации параметров Windows

C#
WinUI 3
.NET

> Об утилите

Tweaker — это современный и мощный инструмент, созданный для того, чтобы дать пользователю полный контроль над операционной системой Windows. Он позволяет отключать телеметрию, оптимизировать службы, кастомизировать интерфейс и повышать общее быстродействие компьютера без глубокого погружения в реестр.

В ближайших обновлениях планируется полноценное слияние с утилитой UWP-Remo, которая станет внутренним модулем для пакетного удаления встроенного мусора и системных приложений в один клик.

Внимание: внесение изменений в системные параметры требует запуска приложения с правами администратора.

> Системные требования

ОС: Windows 10 (1809+)
Библиотека: .NET 10.0+
